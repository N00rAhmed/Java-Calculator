# Java Calculator

## Run Java Calculator

### Change Directory
Go into the correct directory:
```
cd firstjavaproject/src/firstjavaproject
```

### Compile Java Program
```
javac Main.java
```

### Run Java Program
```
java Main.java
```

## Unit Testing

### Run Unit Tests
